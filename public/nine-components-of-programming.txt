The Nine components of programming

1. input / output   -- information into and out of your program

    x = document.all.getElementById('myText01').value   // example input

    document.getElementById('myDiv01').innerHTML = x // example output


2. variables  -- store stuff

   x = 3 
   myY = 'Fred' 

3. loops    --repeat stuff

   for (x=0; x < 3; x++){   // example for loop 

   }

4. decisions   --decide stuff based on stored stuff

   if(x==3) {}

   if (x <= 3){}  else {}

   if ((x > 3) && (x < 9))  {}  else {}

5. functions     -- named group of code that can be called multiple times

   function myFun() { }  // defines a group of code

   myFun()  // activates the function

6. Arrays -- a numbered list of variables

   x[0] = 3;    
   x[1] = 6; 

7. Objects   -- a named list of variables

  x = {age:20, name:'fred'}   // sets object

  // uses object
  x.age    
  x.name

8. events --trigger computer code

   onClick(), onMouseOver(), onBlur(), onFocus() about 20 learn them

9. Classes  -- groups variables and functions to make programming easier
     methods    --> functions
     properties --> variables
